Quick Troubleshooting

Most common issues and how to fix them.

Make sure you clicked the Upload Image button (not the page background). Try refreshing the page. If you are using Live Server, ensure the browser allows file selection.

Use a clear photo (no blur). Ensure the file is JPG/PNG. If the image is too dark or cropped, the analysis may fail or confidence may be low.

Low confidence usually means the image quality is poor (glare, shadows, angle, low resolution). Retake the image using the checklist below and upload again.

The PDF report is generated only after analysis completes. If you are using a demo mode, the report button may show without creating a real PDF. In backend integration, the report will be enabled only after the server creates it.

Accuracy Assistant (Before You Upload)

Follow this checklist to get the best AI accuracy.

Lighting

Use bright, even light. Avoid flash glare and strong shadows.

Full Note

Capture the full note with corners visible. Do not crop.

Sharp Image

Keep camera steady. Use higher resolution for clear security features.

Simple Rule
If text/details are not readable to you, the AI will also struggle.

Understanding Your Result

What “FAKE” indicates

The system found strong signs that the note may be counterfeit. Do not circulate it. Use manual checks and verify via official channels if needed.

What “Confidence” means

Confidence is how strongly the AI model matches the image to learned patterns. Low confidence usually means poor image quality or unclear security features.

Note: This tool supports verification and awareness. For final confirmation, consult official channels.
